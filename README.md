# Color Swatch Tool

Run: python colorswatch-tool.pyw

Features:
* Click to get the details of any TclTk color
* Works with Python 2.7 and 3.4+
* Requires Tk (for Tkinter)

Future work:
* Sort options

Known issues:
* Python 3.4 for Windows incorrectly renders certain colors using HTML rather than TclTk RGB values. These include grey/gray, green, purple and maroon. Tracking via http://bugs.python.org/issue23982
